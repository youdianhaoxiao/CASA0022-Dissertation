Fitness club

This project works in HDRP, URP and Standard Render Pipeline.
To install HDRP click on FitnessClub(HDRP).unitypackage in HDRP_URP_support folder.
To install URP click on FitnessClub(URP).unitypackage in HDRP_URP_support folder.

Dont forget to install HDRP in your project and place HDRenderPipelineAsset in Scriptable Render Pipeline Settings in Graphics tab of Project Settings.

Project includes (46) high quality assets, created for AAA quality visuals.
High quality textures - most 2048 pixels, and some 1024 pixels (for smaller assets) (Albedo, Mask and Normal map)
Most models have 2 LODs

Triangle count for models depends on complexity of the mesh, highest one being 9708 triangles.
For more information about triangle count please read documentation.
